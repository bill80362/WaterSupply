## 說明  

加水站營收系統，用於KEY入每台機子的每月營收，並由系統產生報表和圖表，提供快速檢視。

yarn add  
yarn start

。後端使用php來接應DB  
/water  
。DB是MYSQL  
/water/config/database_config.php
